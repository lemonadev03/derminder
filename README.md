# Derminder

A mobile-optimized PWA for tracking daily skincare and health routines.

## Features

- **AM/PM Toggle** - Switch between morning and evening routines with a smooth animated toggle
- **Day-Aware Schedules** - Face evening routine changes based on day of week
- **Three Sections** - Face, Scalp, and Orals organized in beautiful cards
- **PWA Ready** - Install on your phone's home screen for quick access
- **Dark Theme** - Easy on the eyes with gradient backgrounds

## Routines

### Face
- **Morning**: Azelaq, Nioclean
- **Evening (Mon, Wed, Fri, Sun)**: Iloderm, Glycoa, Nioclean
- **Evening (Tue, Thu, Sat)**: Exfol, Glycoa, Nioclean

### Scalp
- **Both AM & PM**: Soapfree, Seskavel, Minoxidil

### Orals
- **Morning**: Iraltone
- **Evening**: Doxy, Minoxidil, Stercia, Iraltone, Skizin

## Tech Stack

- Next.js 16
- Tailwind CSS 4
- Shadcn UI
- PWA with Service Worker

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) on your phone or browser.

## Installing as PWA

1. Open the app in your mobile browser
2. Tap the share button (iOS) or menu (Android)
3. Select "Add to Home Screen"
4. Enjoy quick access to your routines!
